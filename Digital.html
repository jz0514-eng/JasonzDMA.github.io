Digital

Work created with softwares like Adobe Photoshops and Adobe Illustrator.
Dual Personality - 2024

This artwork is an abstract drawing of a person with two different style and design on each half. This work is created in Adobe Illustrator by take two half two different person and merging them into one to create this vector drawing of it.

Chimera - 2024

This artwork is image of a abstract creature that resemble different creature like a horse and polar bear. This work is created in Adobe Photoshop by merging many image into one and adding abstract color on top.

Three legged Dancer - 2024

This artwork is an abstract image of a dancer with third leg with a unique filters above. This work is created in Adobe Illustrator by creating an vector image of a dancer with some editing in Photo shop , and experimenting with a self created stamp to apply the filters above.

Vector Dog - 2024

This artwork is a vector drawing of a dog. This work is created in Adobe Illustrator by experimenting with basic tool to create a vector image,

This artwork was created when I first started in the Digital Media Art field, and it holds a special meaning to me, as it was a project that shows how you could use simple shapes to recreate an image that resembles a dog.

Eruption - 2024

This artwork is a series of image depicting a eruption of a geyser. This work is created in Adobe Photoshop by experimenting with basic tool like the brush tool and combining them into one image.

Recursion - 2024

This artwork is a abstract glitch art of a image of the same object overlaping each other in different angel and place. This work is created in Adobe Photoshop by stacked many different images of the same object on different layer, and experimenting with different layer setting.
