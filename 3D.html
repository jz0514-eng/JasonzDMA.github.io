3D

Work created within 3D softwares like Blender.
Campfire

This artwork is a 3D scene depicting a campfire with related object like smores and and soda can with stone floors, creating a outdoor camping scene. This work is created in Blender using basic shapes like cubes, and used many preset for texture.

Beace Day

This artwork is a 3D scene depicting a beach with related objects like a beach umbrella and a sand bucket and a toy creature. This work is created in Blender using more techniques like editing mode to create my shapes, and played with texture's setting

Night run

This artwork is a 3D scene depicting a character jogging at a park with night light. This work is created in Blender using more advance techniques like sculping and texture painting, as well as played with texture map.
